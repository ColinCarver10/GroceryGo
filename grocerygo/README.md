## Getting Started

### Environment Variables

Create a `.env.local` file in the root directory with the following variables:

```bash
# Instacart API Configuration
INSTACART_API_KEY=your_instacart_api_key_here

# Supabase Configuration
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

**Getting an Instacart API Key:**
1. Visit [Instacart Connect](https://connect.instacart.com/)
2. Sign up for a developer account
3. Create an application to get your API key

### Running the Development Server

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Features

### Instacart Integration

GroceryGo includes seamless integration with Instacart for ordering groceries directly from your meal plan:

- **One-Click Ordering**: Convert your meal plan shopping list to an Instacart order with a single click
- **Automatic Conversion**: Grocery items are automatically formatted with quantities and units for Instacart
- **Link Back**: Instacart orders include a link back to your meal plan for easy reference
- **24-Hour Expiry**: Shopping links are valid for 24 hours

The integration uses the Instacart Connect API to create shopping lists that users can complete on Instacart's platform.
